GeoCalc SENA - Algoritmo de Figuras Geométricas

GeoCalc es una aplicación web interactiva desarrollada como parte del programa de formación Análisis y Desarrollo de Software del SENA. El proyecto transforma un algoritmo lógico de cálculo geométrico en una herramienta digital moderna, visual y funcional.

🚀 Características

Cálculo de Áreas: Soporte para cuadrado, triángulo, círculo y rectángulo.

Cálculo de Volúmenes: Soporte para cubo y prisma recto.

Interfaz Interactiva: Visualización dinámica de figuras mediante SVG.

Historial de Cálculos: Registro en tiempo real (consola) de los resultados obtenidos durante la sesión.

Hoja de Referencia: Tabla técnica integrada con las fórmulas matemáticas aplicadas.

Diseño Responsivo: Optimizado para dispositivos móviles y escritorio usando Tailwind CSS.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica del proyecto.

Tailwind CSS: Diseño moderno y responsivo.

JavaScript (Vanilla): Lógica del algoritmo, manipulación del DOM y gestión de estados.

SVG: Representación gráfica vectorial de las figuras geométricas.

📖 Estructura del Proyecto

El proyecto sigue una estructura modular que incluye:

Introducción: Contextualización del problema y objetivos.

Calculadora (Algoritmo): El núcleo funcional para la entrada de datos y procesamiento.

Fórmulas: Una sección de consulta técnica (Sheet) para transparencia de los cálculos.

Conclusión: Reflexión sobre la eficiencia de la programación en la resolución de problemas matemáticos.
